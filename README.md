## 项目概述
本作业基于 MuJoCo 物理引擎与 Model Predictive Control (MPC) 框架，开发了一个简化汽车模型的仪表盘系统，并实时显示车辆的运动状态信息，包括车速、转速、油量等。通过本实验，学生可以加深对 MuJoCo 数据结构、物理仿真流程及实时渲染机制的理解，掌握如何将仿真数据转化为可视化效果。

## 项目功能

### 主要功能：
- **车速仪表盘**：显示汽车的当前速度（单位：km/h）。
- **转速仪表盘**：显示发动机的转速（单位：RPM）。
- **油量显示**：实时显示汽车的剩余油量百分比。
- **加速度**：实时显示汽车的加速度信息。
- **温度显示**：模拟发动机温度，随着转速变化。
